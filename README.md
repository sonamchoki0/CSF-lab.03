# CSF101 Lab 03 - Control Structures and Functions

## Aim of the Lab

To develop a clear understanding of control structures in Python by implementing conditional statements, loops, break and continue statements, and to learn how to define and invoke functions for solving programming problems in a structured and efficient manner.

## List of Programs with Brief Explanation

### Part A: Conditional Statements

**q1_positive_negative_zero.py** - Takes a number as input and checks whether it is positive, negative, or zero using if-elif-else.

**q2_even_odd.py** - Takes an integer input and checks if it is even or odd using the modulo operator (%).

**q3_largest_of_two.py** - Takes two numbers as input and compares them to find which one is larger.

**q4_largest_of_three.py** - Takes three numbers as input and uses nested if-else statements to find the largest among them.

**q5_marks_to_grade.py** - Takes marks as input and assigns grades: A for 80 and above, B for 60-79, C for 40-59, Fail for below 40.

**q6_menu_calculator.py** - Displays a menu with options for addition, subtraction, multiplication, and division. Performs the selected operation based on user input.

### Part B: Loops (for and while)

**q7_for_loop_1_to_10.py** - Uses a for loop to print numbers from 1 to 10.

**q8_even_numbers_1_to_20.py** - Uses a for loop with step size 2 to print all even numbers from 1 to 20.

**q9_sum_first_10_natural.py** - Uses a for loop to calculate the sum of first 10 natural numbers (1 to 10).

**q10_multiplication_table.py** - Takes a number as input and uses a for loop to print its multiplication table from 1 to 10.

**q11_while_loop_1_to_5.py** - Uses a while loop to print numbers from 1 to 5.

**q12_sum_1_to_n_while.py** - Takes n as input and uses a while loop to calculate the sum of numbers from 1 to n.

**q13_input_until_zero.py** - Uses an infinite while loop that keeps taking numbers as input until the user enters 0, then breaks the loop.

### Part C: Break and Continue

**q14_break_at_6.py** - Prints numbers from 1 to 10 but uses the break statement to stop the loop when the number reaches 6.

**q15_continue_skip_5.py** - Prints numbers from 1 to 10 but uses the continue statement to skip printing the number 5.

**q16_search_in_list_break.py** - Searches for the number 8 in a predefined list and uses break to stop the loop as soon as the number is found.

### Part D: Functions

**q17_hello_world_function.py** - Defines a simple function called greet() that prints "Hello, World!" and then calls it.

**q18_square_function.py** - Defines a function that takes a number as parameter and returns its square.

**q19_add_two_numbers_function.py** - Defines a function that takes two parameters and returns their sum.

**q20_even_odd_function.py** - Defines a function that checks if a number is even or odd and returns the result.

**q21_factorial_function.py** - Defines a recursive or iterative function to calculate the factorial of a given number.

**q22_multiplication_table_function.py** - Defines a function that takes a number and prints its multiplication table.

### Part E: Integrated programs

**q23_even_odd_1_to_n.py** - Defines a function to check even/odd, then uses a loop to print numbers from 1 to n with their even/odd status.

**q24_menu_calculator_with_functions.py** - Creates separate functions for add, subtract, multiply, and divide. Uses a loop to keep showing the menu until the user chooses to exit.

## How to Run

1. Open any .py file in VSCode
2. Click the Run button or press F5
3. Enter the required inputs when prompted
